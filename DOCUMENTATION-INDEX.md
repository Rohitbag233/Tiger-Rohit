## üîç Quick Reference

### Important URLs
- **Development**: http://localhost:5173/Tiger-Rohit/
- **Production**: https://rohitbag233.github.io/Tiger-Rohit/
- **Admin Panel**: Add `#/admin` to any URL
- **Supabase**: https://app.supabase.com/project/icjahaocvwrvrsilpqwy

### Key Commands
```bash
npm install          # Install dependencies
npm run dev         # Start development
npm run build       # Build for production
npm run preview     # Preview build
npm run deploy      # Deploy to GitHub Pages
```

### Important Files
- `src/App.jsx` - Main application
- `src/pages/` - All pages
- `src/components/` - Reusable components
- `src/styles/index.css` - Global styles
- `vite.config.js` - Build configuration
- `.env.local` - Environment variables

### Common Tasks
- **Update content**: Use admin panel at `/admin`
- **Change colors**: Edit `src/styles/index.css`
- **Add new page**: Create in `src/pages/`, add route in `App.jsx`
- **Database changes**: Edit in Supabase dashboard
